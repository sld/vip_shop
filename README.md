# README

Online shop for VIPs.

## Requirements

* Rails 4.0.0

* Postgresql 9.1.9


## How to run

1. git clone
2. Add database.yml into /config folder
3. bundle install
4. rake db:create && rake db:schema:load
5. rake db:seed 
6. rails server
7. Sign in with email:"email@example.com" password:"password"
